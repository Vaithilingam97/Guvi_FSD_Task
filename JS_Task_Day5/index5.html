Types of Functions

There are 4 types of functions in Javascript.They are

  • 1. Pure Function
  • 2. Anonymous Function
  • 3. IIFE Function
  • 4. Arrow Function

1. Pure Function

A pure function is a function that always returns the same result given the same inputs, and does not modify any external state. In other words, it does not have any side effects.

Syntax:
function (x, y) {
return x + y;
}

2. Anonymous Function

A function without a name is called Anonymous Function. To call this function it need to be stored as a Variable. There are always invoked using variable name.

Syntax:
let add = function(a, b) {
return a + b;
}

3. IIFE Function

IIFE stands for Immediately Invoked Function Expression. It is a common design pattern in JavaScript that involves wrapping an anonymous function in parentheses and immediately invoking it.

Syntax:
(function() {
// code to be executed
})();

4. Arrow Function

Arrow functions are a feature in ES6 that allow for a more concise syntax for writing functions. They have a few differences compared to traditional function declarations, including:

  • Arrow functions are always anonymous.
  • Arrow functions do not have their own "this" keyword, and instead inherit the "this" value from their enclosing context.
  • Arrow functions cannot be used as constructors and do not have a "prototype" property.

    • Syntax:
    const add = (a, b) => {
    return a + b;
    }

    If the function only has one expression, the curly braces and return keyword can be omitted.

    const add = (a, b) => a + b;
