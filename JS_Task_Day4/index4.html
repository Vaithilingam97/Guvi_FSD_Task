Question 1 : How to Compare two JSON have the same properties without order?

    a. let obj1 = {name: "Person 1",age : 5},
    b. let obj2 = {age : 5,name: "Person 1"},

Answer 1 :

    Method 1: Using Javascript :


JSON.stringify(obj1) === JSON.stringify(obj2); // false

    For JSON.stringify(), the order matters.So if the key-value pair are ordered differently in the two objects but are the same, it will return false.



    Method 2: Using Lodash :


_.isEqual(obj1,obj2); // true

    Properties order doesn't matter in Lodash "isEqual", it will return true as along as the key-value pair exists.
